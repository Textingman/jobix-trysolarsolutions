# Society Mortgage

**Your AI Voice Guide Through the Mortgage Process**

Society Mortgage is an AI-powered voice platform that guides homebuyers and homeowners through every step of the mortgage journey — from pre-qualification to closing — with honest, unbiased education.

🌐 **Domain:** [trysocietymortgage.com](https://trysocietymortgage.com)

## What It Does

- **AI Voice Mortgage Guidance** — Ask anything about home loans, rates, approvals, refinancing, and more
- **Step-by-Step Walkthrough** — The AI guides users through every stage of the mortgage process
- **Unbiased Education** — No lender commissions, no conflicts of interest — just honest answers
- **24/7 Availability** — Always available, no appointments needed

## Pages

- `/` — Home page with hero, features, how-it-works, and Q&A preview
- `/about` — Mission, values, and what the AI covers
- `/signup` — Lead capture form to connect with the AI agent
- `/verify` — Phone number verification flow
- `/privacy` — Privacy Policy
- `/terms` — Terms of Service

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
